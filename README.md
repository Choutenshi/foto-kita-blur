# 📸 Foto Kita Blur

The second project of mine, it's inspired from a song with the same title. You just hover your hands doing peace signs and boom, it's all blurry.

## Features

- 📷 Live webcam
- 🌫️ Gaussian blur
- ⌨️ Press B to toggle blur
- ✌️ Peace sign detection (WIP)
- ❌ Press Q to quit

## Installation
```pip install -r requirements.txt```

## Run
```python src/main.py```
